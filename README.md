# GCN-Align

Code of the paper: ***Cross-lingual Knowledge Graph Alignment via Graph Convolutional Networks***.

Initial datasets are from [JAPE](https://github.com/nju-websoft/JAPE).

# Environment

* python>=3.5
* tensorflow>=1.10.1
* scipy>=1.1.0
* networkx>=2.2

# Running

```
python train.py --lang zh_en
```
